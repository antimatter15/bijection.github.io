Snakt

Snakt is a video app that lets you reuse parts of other peoples videos while making sure advertising money is split fairly. Balancing editing and viewing is usually hard, but our novel UI makes it easy.

Heres a web demo of the core UI paradigm. It's written in js and rendered on canvas with no 3rd party libraries. You can drag with your mouse, press the left and right arrow keys, and scroll.

Here are some videos of the actual app. 'Coming soon' is slightly outdated — Snakt is now on the app store, though you need an invite code to use it.

Tesseract.js

Tesseract.js is an in-progress javascript OCR library that can handle 62 languages — the previous state of the art was one language, badly. The main page has a 2d projection of a tesseract, an editable demo, a JSON explorer, and some nice animations when you make the page thin.

Check it out here.

My classes

Every year I make a page for myself that has links to all of my classes. It's understandably not my favorite place to visit, so I try to make it extra upbeat. You can drag the little rays around, and when you click a class they all fly to left, as if you're travelling toward the class website in cyberspace.

Made with boring ol' html and javascript. You can check it out here.

BrassDoge.com

MIT graduates are easy to spot — they're the ones wearing MIT's enormous 'brass rat' class ring. The brass doge adressesses the brass rat's dual problems of costing like $800 and not having enough doge. If you hover over the buy now button on the website a disco party starts, which hopefully encourages you to buy a ring. The payment form does that cool displaying your credit card type as you type your credit card number thing.

The ring itself is modeled in Rhinoceros, and lox-wax cast by shapeways. The website is made from scratch with the exception of three.js for the ring display.

Check it out here.

Smoakapp.com

An anonymous social media app where content spreads across the social and geographic graphs simultaneously — now defunct. Smoke follows your mouse to reveal the text on the website.

The smoke is rendered onto an html canvas element with a little smoke making library I wrote. The key to producing a reasonable smoke effect it to use tiny images upscaled a lot on render. Even small images make the whole thing pretty slow.

Check it out here.

The animation at the top of this page

The header of this page is a particle based fluid simulation. Fluid simulations don't usually make good decoration, because good fluid simulations should converge to a stable state — peacful waters and all that. To fix this, the viscosity calculation is purposefully broken. When particles interact, they generate excess angular momentum and move around all wild-like.

All of the particles are draggable (try it!), and the parameters of the simulation can change in real time - right now they're set to chill out as you scroll down. Everything here is from fram scratch — no physics or animation libraries.
